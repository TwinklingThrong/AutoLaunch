# AutoLaunch

AutoLaunch 是一款基于 Python + PySide6 的 **Windows 桌面“空间管理/一键启动”工具**。  
你可以把「工作」「学习」「娱乐」等不同场景下需要打开的一组应用、文档、文件夹、网页保存成“空间”，之后一键切换整套环境。

---

## 功能特性

- **空间分组管理**
  - 支持创建多个“空间分组”（例如：工作 / 学习 / 娱乐 / 副业）。
  - 每个分组中可以加入：可执行程序（`.exe`）、任意文件、文件夹、浏览器程序 + 启动网址列表。

- **一键启动 / 关闭空间**
  - 点击“启动此空间”：自动启动分组内配置的所有应用和文件。
  - 可选：启动时关闭“不在当前空间且不在白名单”的应用，保持桌面干净。
  - 点击“关闭此空间”：尝试关闭当前空间中的应用进程以及常见文档/文件夹的宿主程序。

- **从当前环境一键建空间**
  - 扫描当前桌面上可见（包括任务栏最小化）的窗口，自动识别对应的应用。
  - 使用 `pywinctl` + `psutil` 尽量记录窗口的位置、大小、状态。
  - 对通过 AutoLaunch 启动的 Chrome / Edge，尽可能读取当前打开的标签页 URL，并保存到空间中，方便下次恢复。

- **应用扫描**
  - 提供“应用扫描”对话框，从指定目录（默认 Program Files）自动扫描 `.exe`。
  - 自动过滤卸载程序、安装器、系统应用，仅保留看起来像“主程序”的可执行文件。
  - 按图标、目录深度、文件大小等多维度排序，方便选出真实主程序。

- **拖拽添加**
  - 可以直接把文件或文件夹拖到主窗口右侧区域，快速加入当前空间。
  - 支持拖入快捷方式（`.lnk`），内部会尝试解析到真实目标路径。

- **右键菜单集成**
  - 在资源管理器中右键文件 / 文件夹 / 盘符，选择“添加到 AutoLaunch 空间”，可快速将目标加入某个空间。
  - 在桌面空白处右键，选择“保存当前窗口到 AutoLaunch 空间”，可一键用当前桌面环境创建新空间。

- **开机自启**
  - 通过勾选“开机自启 AutoLaunch”，在当前用户的注册表 `Run` 项中写入开机启动项。
  - 再次取消勾选，会移除对应自启配置。
